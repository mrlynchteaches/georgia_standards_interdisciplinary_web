# georgia_standards_interdisciplinary_web
This is a website that will allow teachers to input their upcoming standards and to find connections with other courses in the Georgia Standards of Excellence catalog. This will also allow teachers to see how strong the connection to the other content is with low, medium, and high indicators. The results display in an interactive web format. 
Currenly operating using the downloaded JSON format from the SuitCase for Georgia Standards. Would love to learn how to make adjustments to the website in order to have a live update with URI or API calls; but I kept getting stuck and some of the courses would not show up under certain disciplines/subjects. For example, math only had 2 courses listed, when there are 20 for 9-12 grades alone.
